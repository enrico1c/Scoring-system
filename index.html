This site needs a Python backend to fetch market data. Deploy it for free in 2 minutes:

  1. Go to render.com and sign in with GitHub
  2. Click New + → Web Service
  3. Select the enrico1c/Scoring-system repository
  4. Render auto-reads render.yaml — click Create Web Service
  5. Wait ~3 min for build, then copy the URL (e.g. https://uar-backend.onrender.com)
  6. Paste it below and click Save — the setting is remembered in your browser
